/application/

This directory contains all the files for your application. Typically, this is the only directory you will be adding and editing files.

/media/

This directory typicaly contains all the files for your application such as images, flash, css and javascript files.

/core/

This is the framework system files. You should never have to edit these. When upgrade your framework, this is the directory that need to be upgraded.
