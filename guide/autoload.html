Auto Load

Instead of using $this->load->library() or $this->load->helper() and $this->load->model() in your controllers, you can have them autoloaded. Then it will be available on every controllers.

/application/configs/autoload.php

/**
 * Libraries
 *
 * Auto loaded application libraries 
 *
 * @access public
 * @value array
 */ 
$autoload['libraries'] = array('mylib', 'myotherlib');

/**
 * Helpers
 *
 * Auto loaded application udf scripts 
 *
 * @access public
 * @value array
 */ 
$autoload['helpers'] = array('myhelpers', 'myscripts');

/**
 * Models
 *
 * Auto loaded application models 
 *
 * @access public
 * @value array
 */ 
$autoload['models'] = array('mymodel', 'other_model');

Just add your libraries or helpers or models name to the auto-load arrays.

If you want mixed using alias name on auto-load library or model, use an array:

/**
 * Libraries
 *
 * Auto loaded application libraries 
 *
 * @access public
 * @value array
 */
$autoload['libraries'] = array(
					array('mylib', 'myalias'),
					array('myotherlib', 'otheralias'),
					'customlib'
				);
	
/**
 * Models
 *
 * Auto loaded application models 
 *
 * @access public
 * @value array
 */
$autoload['models'] = array(
					array('mymodel', 'my_alias'),
					array('myothermodel', 'otheraliasmodel'),
					'pagemodel'
				);
To keep the application performance, do not auto-load libraries or helpers that not regularly used.
