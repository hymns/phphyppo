Custom Database Library

If want to use something other than the built-in PDO database library, do the following:

Assuming your database library is named CustomDB:

  • Copy db.ActiveRecord.php from /path/to/core/plugins/db.ActiveRecord.php to /path/to/application/plugins/db.CustomDB.php
  • Edit the file, rename the class name from ActiveRecord to CustomDB.
  • Add or edit functionality for your suitable.
  • Edit /path/to/application/configs/database.php, change the $config['db_plugin'] value from ActiveRecord to CustomDB.
